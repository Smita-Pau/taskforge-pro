TASKFORGE PRO

A full-stack task and project management web application built for engineering teams.

----------------------------------------
FEATURES
----------------------------------------
- User authentication (JWT login system)
- Project creation and management
- Task creation, update, and tracking
- Team member management
- Dashboard with real-time project progress
- Role-based access control

----------------------------------------
TECH STACK
----------------------------------------
Frontend:
- React (Vite)
- Tailwind CSS
- Axios

Backend:
- Node.js
- Express.js

Database:
- MongoDB Atlas

----------------------------------------
DEPLOYMENT
----------------------------------------
Frontend: Vercel
Backend: Railway
Database: MongoDB Atlas

----------------------------------------
HOW TO RUN LOCALLY
----------------------------------------
1. Clone repository
2. Install dependencies: npm install
3. Setup .env file
4. Run backend: npm run dev
5. Run frontend: npm run dev
